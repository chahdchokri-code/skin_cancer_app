# 🔬 SkinAI — Application de Diagnostic de Lésions Cutanées

> Application Web complète en Python (Flask) intégrant un modèle de Deep Learning **VGG16** pour la détection de lésions cutanées malignes ou bénignes.

![Dashboard](screenshots/dashboard.png)

---

## ✨ Fonctionnalités

- 🔐 **Authentification** — Connexion sécurisée pour les professionnels de santé
- 🩺 **Analyse IA** — Diagnostic instantané Bénin / Malin via VGG16
- 📊 **Dashboard** — Statistiques globales en temps réel
- 📋 **Historique patients** — Consultation de toutes les analyses enregistrées
- 🛡️ **Données locales** — Stockage exclusif en base MySQL locale
